# Bayonetta2Blender2Bayonetta
Import/Export WMB (version 0) models straight from Blender.

| Game Name | WMB | SCR | COL | 
|---|---|---|---|
| Bayonetta 1 | Import/Export | Import | Planned |
| Bayonetta 2 | Import/Export | Import | Planned |
| Vanquish | Import/Export | Import (.LYT) | No ([Requires Havok](https://www.youtube.com/watch?v=lrSaXzGGM8U)) |
| Anarchy Reigns | Untested | Untested | Planned |
| Wonderful 101 | Import/Export | Untested | Planned |
| The Legend of Korra | Untested | Untested | Planned |
| Transformers: Devastation | Import/Export | Untested | Planned |
| Madworld | Planned (Import only, MDB) | No | No |

## Notes about other titles:
- Metal Gear Rising (2013) uses WMB4, this is a very different format from what this plugin targets, please use [MGR2Blender2MGR](https://github.com/Jacky720/MGR2Blender2MGR)
-  Almost all other games use Platinum's improved WMB3 format, the [Nier2Blender2Nier](https://github.com/WoefulWolf/NieR2Blender2NieR) plugin targets these games.
